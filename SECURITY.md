# Security Policy

This is a static educational portfolio project. It should not collect sensitive personal data or store secrets in the repository.

## Supported Scope

- Static HTML, CSS and JavaScript
- Netlify deployment configuration
- Browser-only local progress storage

## Operational Rules

- Do not add API keys or private credentials to the front end.
- Keep the app free of login, chat, payments and unnecessary data collection unless a full privacy review is done.
- Review third-party scripts before adding them.

## Reporting

Open a private GitHub issue or contact the maintainer through the GitHub profile.
