# Deploy Gratuito - Trilha da Palavra SN

## Objetivo

Publicar a Trilha da Palavra SN como site público para teste com família, igreja, escola bíblica e responsáveis fora da mesma rede do computador.

## Plataforma Recomendada

**Netlify** é a opção recomendada para validação pública inicial porque aceita projeto estático em HTML, CSS e JavaScript sem servidor próprio.

## Configuração do Projeto

O arquivo `netlify.toml` define:

- publicação da raiz do projeto;
- ausência de etapa de build;
- cache desativado para evitar que celular ou tablet carregue versão antiga.

## Fluxo de Publicação

1. Acessar o Netlify.
2. Criar uma conta gratuita ou entrar em uma conta existente.
3. Publicar a pasta do projeto.
4. Testar o link público no celular, tablet e computador.
5. Enviar o link final pelo WhatsApp somente depois de validar que o botão "Começar agora" inicia uma rodada.

## Critério de Aceite

- O link abre fora do computador local.
- O site carrega no celular usando internet móvel ou Wi-Fi de outra cidade.
- O botão "Começar agora" inicia a primeira pergunta.
- Depois de responder, aparece o botão "Próxima".
- A tela final aparece ao concluir a rodada.

## Observação

Túneis temporários servem apenas para teste rápido. Para enviar para pessoas em outra cidade, o ideal é usar um link hospedado em plataforma pública.
