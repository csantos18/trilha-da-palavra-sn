# Política de Privacidade e LGPD - Trilha da Palavra SN

## 1. Objetivo

Esta política explica como o projeto Trilha da Palavra SN trata privacidade, segurança e dados no uso do jogo educativo.

O jogo foi planejado para ser usado por famílias, igrejas, escolas bíblicas e responsáveis, com foco em pré-adolescentes de 10 a 12 anos.

## 2. Coleta de Dados Pessoais

O projeto não coleta dados pessoais.

Não são solicitados:

- nome;
- e-mail;
- telefone;
- endereço;
- imagem;
- localização;
- documento;
- idade individual;
- senha;
- conta de usuário.

## 3. Dados Armazenados no Navegador

O jogo pode salvar progresso no navegador do usuário usando `localStorage`.

Essas informações são técnicas e locais, como:

- pontuação;
- medalhas;
- modo jogado;
- progresso por personagem.

Esses dados ficam somente no dispositivo/navegador usado e não são enviados para servidor externo pelo projeto.

## 4. Crianças e Adolescentes

Como o projeto é destinado a público infantojuvenil, o uso recomendado é acompanhado por pais, responsáveis, líderes ou educadores.

O jogo não possui chat, cadastro, ranking público, anúncios, compras internas ou interação direta entre usuários.

## 5. Finalidade do Uso

A finalidade do projeto é educativa:

- apoiar ensino bíblico;
- estimular reflexão sobre escolhas;
- incentivar atitudes práticas no dia a dia;
- oferecer uma experiência segura e simples.

## 6. Compartilhamento de Dados

O projeto não compartilha dados pessoais porque não coleta dados pessoais.

Também não possui integração externa obrigatória para envio de informações.

## 7. Exclusão de Dados Locais

O progresso salvo pode ser apagado limpando os dados do navegador ou removendo o armazenamento local do site.

Como o armazenamento é local, cada dispositivo/navegador possui seu próprio progresso.

## 8. Segurança

Medidas adotadas na proposta atual:

- ausência de login;
- ausência de chat aberto;
- ausência de anúncios;
- ausência de compras internas;
- ausência de coleta de dados pessoais;
- uso local do progresso no navegador.

## 9. Responsabilidade de Uso

O conteúdo deve ser usado como apoio educativo. Para uso em igrejas, escolas ou grupos, recomenda-se revisão prévia por responsável bíblico/pedagógico.

## 10. Atualizações desta Política

Caso o projeto passe a ter servidor, cadastro, analytics, ranking online, painel administrativo ou qualquer integração externa, esta política deve ser revisada antes da publicação.
