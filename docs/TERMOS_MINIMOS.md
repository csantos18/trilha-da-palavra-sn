# Termos Minimos - Trilha da Palavra SN

## 1. Aceite

Ao utilizar a Trilha da Palavra SN, o usuario ou responsavel reconhece que o projeto e uma ferramenta educativa de apoio, voltada para desafios biblicos praticos e uso acompanhado.

## 2. Uso Adequado

O projeto deve ser usado por familias, igrejas, escolas biblicas, professores, lideres ou responsaveis. Para criancas e pre-adolescentes, recomenda-se acompanhamento adulto.

## 3. Conteudo

O conteudo tem finalidade educativa e deve ser revisado pela instituicao, lideranca ou responsavel antes de uso publico amplo.

## 4. Limitacoes

O projeto nao substitui acompanhamento familiar, pedagogico, pastoral ou ministerial. Tambem nao deve ser usado como ranking publico, avaliacao formal ou comparacao entre criancas.

## 5. Dados e Privacidade

O projeto nao solicita cadastro e nao coleta dados pessoais. O progresso pode ficar salvo localmente no navegador do usuario.

## 6. Disponibilidade

Como aplicacao web estatica, a disponibilidade depende da plataforma de hospedagem, conexao de internet e navegador utilizado.

## 7. Alteracoes

Os termos podem ser atualizados caso o projeto receba novas funcionalidades, como painel administrativo, servidor, analytics, cadastro, relatorios ou integracoes externas.
